《Re:Fate》——基于Unity引擎的游戏设计与实现

## 项目简介
《Re:Fate》是一款基于 Unity 引擎开发的2.5D游戏项目，项目围绕角色养成、基础战斗逻辑与游戏系统设计展开，旨在实践完整的游戏开发流程，包括系统设计、玩法实现与基础交互逻辑搭建。
本仓库主要用于代码托管与版本管理，完整项目内容包含游戏核心逻辑实现、基础功能模块以及部分系统设计实现。

## 项目说明
由于仓库仅用于托管源码（Code Only），未包含完整游戏构建文件（Build）及部分资源文件，以保证仓库体积与结构规范。
游戏实机运行效果、玩法展示及界面表现请参考下方截图与演示内容。

## 技术栈
- Unity（游戏引擎）
- C#
- 2.5D游戏开发框架
- NavMesh
- 基础游戏系统设计（角色控制 / 场景交互 / UI逻辑）

## 项目职责（个人项目）
- 完成游戏整体功能设计与模块划分
- 实现角色基础控制与交互逻辑
- 搭建游戏基础场景与摄像机系统
- 编写核心功能脚本（C#）
- 进行玩法测试与迭代优化

## 补充说明
本项目为个人独立开发的学习型游戏Demo，侧重于游戏系统实现与开发流程实践，适用于游戏策划 / Unity相关岗位的项目展示。
